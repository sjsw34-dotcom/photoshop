# docs/landing-page.md — 랜딩페이지 명세 (app/page.tsx)

> 첫 방문자가 "이 사이트가 나에게 맞는지" 30초 안에 판단할 수 있는 페이지. **이 파일은 `app/page.tsx`와 랜딩 전용 섹션 컴포넌트를 구현할 때 참조한다.**

---

## 1. 전체 원칙

- **톤**: 따뜻하고 부담 없고 정돈된. 학원 광고처럼 과장하지 않는다.
- **목표**: 공백기가 있는 학생, 기초가 없는 복학생, 동급생을 따라잡아야 하는 사람이 "아, 이거다" 느끼게.
- **길이**: 스크롤 5-7회 분량. 무한 스크롤 금지.
- **CTA**: 페이지 내 최소 3회 반복 (히어로, 중간, 마지막).

## 2. 섹션 구조 (순서 고정)

### 2.1 히어로 (Hero)

**레이아웃**: 왼쪽 텍스트 + 오른쪽 일러스트 (모바일: 상하)

**CTA**:
- Primary: "무료로 시작하기" → `/lessons/0/what-is-photoshop`
- Secondary: "커리큘럼 먼저 보기" → `/curriculum`

**비주얼**:
- 간단한 라인 일러스트 (포토샵 창 추상화)

### 2.2 공감 섹션 ("이런 분이라면")

**헤드라인**: "혹시 이런 고민이 있으신가요?"

**3개 카드** (가로 배치, 모바일 세로):

1. 📚 **"학교 수업은 이미 진도가 나가고 있는데, 저만 기초가 없어요"**
   - 복학 후 따라잡기가 막막한 학생

2. ⏰ **"공부할 시간이 하루 1시간뿐이에요"**
   - 아르바이트, 다른 수업, 생활 리듬이 있는 학생

3. 📖 **"유튜브 영상은 너무 빠르고, 책은 두꺼워서 안 읽혀요"**
   - 자기 속도로 꼼꼼히 이해하고 싶은 학생

**마무리 한 줄**: "셋 중 하나라도 맞는다면, 이 사이트는 당신을 위해 만들어졌습니다."

### 2.3 학습 철학 ("3가지 약속")

**헤드라인**: "이 사이트의 3가지 약속"

**3개 섹션** (세로 배치, 각 섹션에 아이콘 + 제목 + 본문):

1. 🌱 **기초부터 다시 시작합니다**
   - "마우스 클릭이 뭔지부터 설명합니다. 남들이 당연히 알고 있다고 넘어가는 것들을 하나하나 짚습니다."

2. 🕐 **하루 1시간을 넘기지 않습니다**
   - "각 레슨은 40-60분 분량입니다. 중간에 멈췄다가 다음 날 정확히 그 지점에서 이어갈 수 있어요."

3. 🎥 **동영상 없이도 따라할 수 있습니다**
   - "스크린샷, 단계별 설명, 연습 파일, 체크리스트로 구성했습니다. 내 속도대로 멈추고, 되돌아보고, 다시 해볼 수 있어요."

### 2.4 커리큘럼 미리보기

**헤드라인**: "4단계로 차근차근"

**4개 카드** (2x2 그리드, 모바일 세로):

각 카드 내용:
```
[Level N]
[레벨 이름]
[레슨 개수] · [예상 기간]

졸업하면 할 수 있는 것:
✓ [구체적 능력 1]
✓ [구체적 능력 2]

[주요 키워드 3-5개를 태그로]
```

**카드별 채움**:

**Level 0 · 입문** (10레슨 · 약 2주)
- 졸업 후: 포토샵을 혼자 켜서 사진을 불러오고 크기를 바꿔 저장할 수 있다
- 태그: `화면 구조` `파일 열기` `저장하기`

**Level 1 · 초급** (25레슨 · 약 5주)
- 졸업 후: 레이어를 자유롭게 다루고 간단한 합성과 보정을 할 수 있다
- 태그: `레이어` `선택 영역` `텍스트`

**Level 2 · 중급** (25레슨 · 약 5주)
- 졸업 후: 포스터, 상세페이지, 인물 보정을 혼자 완성한다
- 태그: `마스크` `혼합 모드` `펜 툴` `인물 보정`

**Level 3 · 고급** (20레슨 · 약 4주)
- 졸업 후: 전공 과제와 포트폴리오를 포트폴리오 품질로 완성한다
- 태그: `제품 보정` `목업` `액션` `출력`

**하단 링크**: "전체 커리큘럼 자세히 보기 →" → `/curriculum`

### 2.8 FAQ

**헤드라인**: "자주 묻는 질문"

**5-7개 아코디언**:

1. **Q. 포토샵이 없어도 되나요?**
   A. 포토샵은 별도로 설치해야 합니다. 2번째 레슨에서 Adobe 7일 체험판 설치법을 설명드려요. 학생 할인 구독도 안내합니다.

2. **Q. 정말 동영상 없이 따라할 수 있나요?**
   A. 네. 각 레슨은 스크린샷, 단계별 번호 설명, 키보드 단축키 시각화, 연습 파일, 체크리스트로 구성됩니다. 막히는 지점마다 "이게 안 될 때" 문제해결 박스도 있어요. 첫 레슨을 보시면 감이 올 거예요.

3. **Q. 하루 1시간이 정말 충분한가요?**
   A. 충분합니다. 각 레슨이 40-60분 분량이고, 4단계를 다 마치면 약 3-4개월이 걸립니다. 주말을 빼고 주 5일만 공부해도 100일이 안 걸려요.

4. **Q. 학교 진도를 따라잡을 수 있나요?**
   A. Level 2를 마치면 학과 평균 수준에 근접하고, Level 3를 마치면 전공 과제를 독립적으로 해낼 수 있습니다. 디자인공학과 같은 융합 학과라면 Level 3의 제품 보정/목업 파트가 특히 도움이 됩니다.

5. **Q. 중간에 쉬어도 되나요?**
   A. 네. 진도는 자동 저장됩니다. 다만 너무 오래 쉬면 이전 내용을 잊으니 주 3회 이상을 권장해요. 연속 학습일이 끊어져도 다시 시작하면 됩니다.

6. **Q. 가입해야 하나요?**
   A. 가입 없이 모든 레슨을 이용할 수 있습니다. 진도는 브라우저에 저장돼요. 단, 같은 기기와 브라우저를 계속 사용해야 진도가 이어집니다.

7. **Q. 이 사이트는 무료인가요?**
   A. 네, 현재는 무료입니다.

## 3. 디자인 스펙

### 3.1 컬러
- 배경: `bg-white dark:bg-slate-950`
- 텍스트: `text-slate-800 dark:text-slate-100`
- 강조: `text-orange-600` / `bg-orange-500`
- 보조 배경(카드): `bg-slate-50 dark:bg-slate-900`

### 3.2 타이포
- 한글: `Pretendard` (system fallback: Noto Sans KR)
- 영문: `Inter`
- 헤드라인: `font-bold text-3xl md:text-5xl`
- 본문: `text-base md:text-lg leading-relaxed`
- 섹션 헤딩: `text-2xl md:text-3xl font-bold mb-4`

### 3.3 레이아웃
- 최대 너비: `max-w-6xl mx-auto`
- 좌우 패딩: `px-4 md:px-6 lg:px-8`
- 섹션 상하 여백: `py-16 md:py-24`

### 3.4 애니메이션
- 스크롤 인 애니메이션: 각 섹션 진입 시 fade + translate-y-4
- Framer Motion 대신 CSS `@media (prefers-reduced-motion: reduce)` 존중

## 4. 절대 하지 말 것

- "3일 만에 포토샵 마스터!" 같은 과장
- "취업 보장" 같은 약속
- 무한 스크롤 가로 캐러셀 (접근성 최악)
- 자동 재생 비디오
- 팝업 모달 (뉴스레터 가입 요청 등)
- 이모지 남용 (섹션 제목당 최대 1개)
- em dash (—) 사용 절대 금지

## 5. 접근성

- 모든 이미지에 의미 있는 alt
- 버튼은 `<button>` 또는 `<a>`로 (div 클릭 금지)
- 포커스 링 명확
- 키보드로 모든 CTA 도달 가능
- 색 대비 WCAG AA 이상
- 스크린리더가 섹션 순서대로 읽을 수 있게 `<section>` 태그 활용
