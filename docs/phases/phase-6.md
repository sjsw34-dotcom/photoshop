# Phase 6 — 확장

> **목표**: Level 1-3 레슨을 순차적으로 채우고, AI 도우미를 연결. 첫 공개 이후 지속적으로 진행되는 단계.

> **예상 소요**: 수 주-수 개월 (지속적)

> **시작 전 참조 문서**: `CLAUDE.md`, `docs/writing-style.md`, `docs/lesson-template.md`, `docs/curriculum.md`, `docs/offline-learning-devices.md` (Section 장치 7)

> **선행 조건**: Phase 5 완료 (사이트가 배포 가능한 상태)

---

## 범위 (Scope)

**이 Phase에서 하는 것** (우선순위 순):
1. Level 1 초급 25레슨 작성 (최우선)
2. AI 도우미 구현 (중간 우선)
3. Level 2 중급 25레슨 작성
4. Level 3 고급 20레슨 작성
5. 커뮤니티 Q&A (선택)

**이 Phase의 특징**:
- 한 번에 전부 하지 않는다
- 각 작업은 독립적인 미니 Phase로 진행
- 아들의 실제 학습 진도에 맞춰 우선순위 조정 가능

## 우선순위 결정 기준

다음 순서로 진행 권장:

1. **아들이 Level 0을 다 마쳤는가?** → Level 1 작성 시작
2. **아들이 Level 1 중반에 왔는가?** → AI 도우미 구현 (막히는 지점이 늘어나는 시점)
3. **아들이 Level 1을 다 마쳤는가?** → Level 2 작성
4. **학과 과제 시즌인가?** → Level 3의 디자인공학과 특화 파트 우선
5. **다른 학습자도 보고 있는가?** → 커뮤니티 Q&A 고려

## 주의 사항

### 콘텐츠 작성 관련
- **하루에 레슨 3-5개 이상 작성하지 않기.** 품질 떨어짐. 자가 검증도 부실해짐.
- **Level이 올라갈수록 난이도에 맞게 학습자 기대 수준 조정.** Level 3 학습자는 이미 중급자다. 비유는 줄이고 전문성을 높인다.
- **디자인공학과 특화 레슨(Level 3)**은 마스터가 전공자에게 검수 요청 권장.

### AI 도우미 관련
- **비용 모니터링.** 월간 API 비용 한도 설정.
- **환각 위험.** Claude가 틀린 포토샵 지식을 자신있게 답할 수 있다. 답변 말미에 "확실하지 않으면 공식 문서를 참고하세요" 같은 면책 자동 추가.
- **학습자 데이터 프라이버시.** 질문 내용을 로깅할 경우 고지 필수.

### 장기 운영 관련
- **Phase 6는 Phase 5까지와 달리 "완료"가 모호하다.** 전체 80레슨 완성이 최종 목표지만, 아들이 학습 중인 레벨까지만 채워져 있어도 가치가 있다.
- **실제 사용자(아들) 피드백이 가장 중요하다.** 어느 레슨이 어렵다고 했는지, 어느 단축키가 안 외워진다고 했는지를 기록해 레슨에 반영.
