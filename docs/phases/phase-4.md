# Phase 4 — 초기 콘텐츠 (Level 0, 10레슨)

> **목표**: Level 0의 10개 레슨 MDX 파일을 완성. 학습자가 실제로 사용할 수 있는 첫 번째 레벨 전체를 확보.

> **예상 소요**: 3-5일 (한 번에 다 하지 않는다. 여러 세션에 나눠 진행 권장)

> **시작 전 참조 문서**: `CLAUDE.md`, `docs/writing-style.md` (필독, 매 레슨마다 재확인), `docs/lesson-template.md`, `docs/curriculum.md` (Level 0 부분), `docs/offline-learning-devices.md`

> **선행 조건**: Phase 3 완료

---

## 범위 (Scope)

**이 Phase에서 하는 것**:
- Level 0 레슨 10개 작성
- 각 레슨의 스크린샷 자리(placeholder) 확보 + 구체적 alt 텍스트
- 연습 파일 스펙 정의 (실제 .psd 제작은 마스터 수작업)
- 용어 사전 항목 추가 (레슨에서 등장한 용어)

**이 Phase에서 하지 않는 것**:
- Level 1-3 레슨 (Phase 4의 확장 또는 별도 Phase)
- 실제 스크린샷 이미지 (마스터가 수동 촬영)
- 연습 파일 .psd 생성 (마스터 수작업)

---

## 작업 방식 (중요)

### 한 번에 한 레슨씩 작성

**잘못된 방식**: "레슨 10개 다 써줘" → Claude Code 과부하, 품질 저하, 일관성 무너짐.

**올바른 방식**: 
1. 마스터가 "레슨 1번 써줘" 명령
2. Claude가 1번만 집중 작성
3. 마스터가 검토하고 수정 요청 또는 OK
4. 다음 레슨으로

### 매 레슨 작성 시 자가 점검

작성 후 반드시 `docs/lesson-template.md` Section 7의 체크리스트 확인.

제목: 포토샵이 뭐고 왜 쓰나요

**핵심 메시지**:
- 포토샵은 "디지털 그림 도구"
- 왜 쓰나: 사진 편집, 디자인, 합성, 로고, UI 디자인 등
- "프로그램이 어렵게 느껴지는 건 당연해요. 같이 천천히 해봐요."

**섹션 포함**:

- 단축키: 없음
- 자주 묻는 문제: "포토샵 vs 일러스트레이터 차이?", "무료 대안은?"

### 4.5 레슨 4: `mouse-keyboard-basics` (40분)

제목: 마우스와 키보드 기본 조작

**핵심 메시지**:
- 클릭, 더블클릭, 드래그
- Ctrl/Cmd, Alt/Option, Shift의 의미 (각각이 뭘 도와주는지)
- 우클릭 메뉴
- 스페이스바 + 드래그 = 화면 이동

**일상 비유**:
"Shift 키는 '똑바로 해줘'라고 말하는 거예요. 원을 그릴 때 Shift를 누르면 정원이 되고, 선을 그릴 때 누르면 수평선이나 수직선만 나와요. Alt는 '복사해줘' 느낌이에요. 레이어를 Alt 누른 채 드래그하면 원본은 남고 복사본이 만들어져요."

### 4.10 레슨 9: `save-formats` (50분)

제목: 저장하기 (PSD, JPG, PNG 차이)

**일상 비유**:
"PSD는 레고 블록을 다 따로 보관한 상자, JPG는 조립이 끝난 작품 사진, PNG는 투명 배경이 필요할 때 쓰는 특별한 사진이에요."

**다룰 내용**:
- 저장 (Ctrl+S) vs 다른 이름으로 저장 (Ctrl+Shift+S)

- JPG: 용량 작음, 투명 불가
- PNG: 투명 가능, 용량 큼
- 웹용 저장 (Save for Web) 간단 언급

## 주의 사항

- **`docs/writing-style.md`를 매 레슨마다 재확인.** 하드 룰(em dash 금지, 문장 50자 이내)을 잊지 말 것.
- **한 번에 한 레슨씩.** "레슨 10개 한꺼번에" 명령받으면 "한 번에 하나씩 하는 게 품질에 좋습니다"라고 제안할 것.
- **비유를 창작할 때 한국 학생에게 익숙한 것으로.** `docs/writing-style.md` Section 3.3 비유 라이브러리 우선 참조.
- **설치/유료 이슈**: 포토샵이 Adobe 유료 구독이라는 사실을 숨기지 말 것. 7일 체험판 + 학생 할인 정보 정확히.
- **퀴즈는 암기 질문이 아니다.** 개념 구분, 실제 사용 상황 판단을 묻는다.
- 커밋 메시지 제안: `feat(phase-4): complete Level 0 lessons (10 lessons)`
