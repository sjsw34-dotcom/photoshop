# docs/curriculum.md — 전체 커리큘럼 (80레슨)

> 4단계 80레슨 구성. 각 레벨은 "졸업 조건"을 명확히 가진다. **이 파일은 레슨 MDX를 작성할 때, 커리큘럼 지도 페이지를 만들 때 참조한다.**

---

## 설계 원칙

1. **실습 프로젝트로 레벨 종결.** 각 레벨 마지막에 실제 결과물 과제.
2. **다음 레벨의 선행 조건은 직전 레벨의 실습 프로젝트 완료.**
3. **레슨 1개 = 40-60분.** 넘치면 두 개로 쪼갬.
4. **순차 진행 권장.** 단, 중급부터는 선택적 경로도 허용.

```yaml
---
title: "레슨 제목"
level: 0 | 1 | 2 | 3
order: 1-80
duration: 45
slug: "what-is-photoshop"
prerequisites: ["이전-slug"]
objectives:
  - "이 레슨이 끝나면 할 수 있는 것 1"
  - "이 레슨이 끝나면 할 수 있는 것 2"
keyTerms: ["레이어", "캔버스"]
practiceFile: "/practice-files/lesson-01.zip"
---
```
