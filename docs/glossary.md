# docs/glossary.md — 용어 사전 시드 데이터

> `/glossary` 페이지와 본문 툴팁의 원본 데이터. **이 파일은 용어 사전 페이지를 구현하거나 초기 JSON 데이터를 시드할 때 참조한다.**

---

## 구조

각 용어는 다음 스키마를 따른다.

```typescript
interface GlossaryItem {
  term: string;              // 한글 명칭
  english: string;           // 영문 명칭
  easyDefinition: string;    // 초등학생도 이해할 쉬운 설명 (1-2문장)
  formalDefinition: string;  // 정식 정의 (기술적)
  analogy?: string;          // 일상 비유 (선택)
  relatedLessons: string[];  // 관련 레슨 slug 배열
  relatedTerms: string[];    // 연결된 다른 용어들
  category: 'basic' | 'layer' | 'tool' | 'color' | 'effect' | 'advanced';
}
```

JSON 저장 위치: `lib/glossary.ts` 또는 `content/glossary.json`

## 용어 시드 데이터 (50개 이상)

### 기본 개념 (basic)

**캔버스 (Canvas)**
- 쉬운 설명: 그림을 그릴 도화지예요. 포토샵에서 작업하는 흰색(또는 투명) 영역입니다.
- 정식 정의: 이미지 편집이 이루어지는 작업 영역. 새 문서를 만들 때 크기와 해상도를 지정한다.
- 비유: 실제 스케치북을 폈을 때 보이는 하얀 종이
- 관련 레슨: `new-document`, `interface-tour`

**픽셀 (Pixel)**
- 쉬운 설명: 이미지를 이루는 아주 작은 네모 칸 하나예요. 모자이크 타일 한 개라고 생각하면 됩니다.
- 정식 정의: 디지털 이미지를 구성하는 최소 단위. 색 정보 하나를 담는 사각형.
- 비유: 모자이크 벽화의 타일 한 장
- 관련 레슨: `what-is-photoshop`, `new-document`

**해상도 (Resolution)**
- 쉬운 설명: 이미지가 얼마나 촘촘한지예요. 타일이 촘촘할수록 선명해요.
- 정식 정의: 단위 길이당 픽셀 수. 인쇄물은 300dpi, 웹용은 72dpi가 표준.
- 비유: 모자이크 벽화를 만들 때 타일 크기. 작은 타일 많이 쓸수록 세밀한 그림.
- 관련 레슨: `new-document`, `export-for-print`, `export-for-web`

**DPI (Dots Per Inch)**
- 쉬운 설명: 1인치(2.54cm) 안에 점이 몇 개 있는지 나타내는 숫자예요.
- 정식 정의: 인쇄 해상도 단위. 높을수록 선명하지만 파일 크기 커짐.
- 관련 레슨: `new-document`, `export-for-print`

**래스터 (Raster)**
- 쉬운 설명: 픽셀(네모 칸)로 이루어진 이미지예요. 사진, JPG가 래스터입니다.
- 정식 정의: 픽셀 격자로 표현되는 이미지 방식. 확대 시 계단 현상 발생.
- 관련 레슨: `what-is-photoshop`, `pen-tool-straight`

**벡터 (Vector)**
- 쉬운 설명: 점과 선으로 이루어진 그림이에요. 아무리 키워도 깨지지 않아요.
- 정식 정의: 수학적 경로로 표현되는 이미지. 확대해도 품질 유지.
- 관련 레슨: `pen-tool-straight`, `vector-mask`

**RGB**
- 쉬운 설명: 빨강(R), 초록(G), 파랑(B) 세 가지 색을 섞어 모든 색을 만드는 방식이에요. 모니터에서 쓰는 색 방식.
- 정식 정의: 가산 혼합 색 모델. 디지털 화면, 웹용 이미지의 표준.
- 관련 레슨: `new-document`, `color-space-print`

**CMYK**
- 쉬운 설명: 인쇄할 때 쓰는 색 방식이에요. 청록, 자홍, 노랑, 검정 네 가지 잉크로 색을 만들어요.
- 정식 정의: 감산 혼합 색 모델. 인쇄물의 표준 색 공간.
- 관련 레슨: `color-space-print`, `export-for-print`

**DPI vs PPI**
- 쉬운 설명: DPI는 인쇄 때, PPI는 화면 표시 때 쓰는 단위예요. 포토샵에서는 대부분 섞어 씁니다.
- 정식 정의: DPI는 Dots Per Inch, PPI는 Pixels Per Inch. 엄밀히 다르지만 혼용됨.
- 관련 레슨: `new-document`

### 레이어 (layer)

**레이어 (Layer)**
- 쉬운 설명: 투명한 셀로판지 여러 장을 겹친 것과 같아요. 각 장에 그림을 그리고 겹치면 합쳐져 보여요.
- 정식 정의: 독립적으로 편집 가능한 이미지 층. 순서, 불투명도, 혼합 모드로 결과 제어.
- 비유: 만화에서 배경, 인물, 말풍선을 각각 다른 투명 필름에 그린 것
- 관련 레슨: `layer-concept`, `layer-create-delete`, `layer-order`

**레이어 그룹 (Layer Group)**
- 쉬운 설명: 레이어들을 폴더 안에 모아놓은 거예요. 지저분한 레이어 패널을 정리할 때 씁니다.
- 정식 정의: 여러 레이어를 묶어 단위로 이동/보임 제어가 가능한 컨테이너.
- 관련 레슨: `layer-group`

**불투명도 (Opacity)**
- 쉬운 설명: 레이어가 얼마나 투명한지 정하는 값이에요. 0%면 안 보이고, 100%면 완전히 보여요.
- 정식 정의: 레이어 전체의 투명도. 레이어 스타일 포함.
- 관련 레슨: `layer-opacity-fill`

**채우기 (Fill)**
- 쉬운 설명: 레이어 내용물만 투명하게 하고, 효과(그림자 등)는 남겨두는 값이에요.
- 정식 정의: 레이어 픽셀의 투명도. 레이어 스타일은 영향받지 않음.
- 관련 레슨: `layer-opacity-fill`, `layer-style-basics`

**혼합 모드 (Blend Mode)**
- 쉬운 설명: 위 레이어가 아래 레이어와 어떻게 섞이는지 정하는 방식이에요.
- 정식 정의: 두 레이어의 픽셀 값을 수학적으로 결합하는 알고리즘. Multiply, Screen, Overlay 등.
- 관련 레슨: `blend-mode-basics`, `blend-mode-overlay`

### 마스크 (layer)

**레이어 마스크 (Layer Mask)**
- 쉬운 설명: 종이에 창문을 뚫어 그 부분만 보이게 하는 것과 같아요. 검은색은 숨김, 흰색은 보임이에요.
- 정식 정의: 레이어에 적용되는 8비트 흑백 알파 채널. 비파괴 방식으로 일부만 보이게 함.
- 비유: 스텐실 종이
- 관련 레슨: `layer-mask-intro`, `mask-edit-deep`

**클리핑 마스크 (Clipping Mask)**
- 쉬운 설명: 아래 레이어 모양으로 위 레이어를 잘라내는 거예요. 쿠키 틀로 반죽을 찍는 것과 같아요.
- 정식 정의: 아래 레이어의 투명도를 기준으로 위 레이어가 보이게 하는 방식.
- 비유: 쿠키 틀
- 관련 레슨: `clipping-mask`

**벡터 마스크 (Vector Mask)**
- 쉬운 설명: 펜 툴로 그린 깔끔한 경계로 레이어를 자르는 마스크예요.
- 정식 정의: 벡터 패스를 이용한 마스크. 해상도 무관하게 선명.
- 관련 레슨: `vector-mask`

### 선택 영역과 도구 (tool)

**선택 영역 (Selection)**
- 쉬운 설명: 작업하고 싶은 영역을 지정하는 거예요. 점선으로 표시돼요.
- 정식 정의: 편집 작업이 적용될 픽셀 범위.
- 관련 레슨: `selection-rect-oval`, `selection-lasso`, `selection-magic-wand`

**올가미 (Lasso)**
- 쉬운 설명: 원하는 모양대로 자유롭게 그려 영역을 선택하는 도구예요.
- 정식 정의: 자유 곡선으로 선택 영역을 만드는 도구. 다각형/자석 올가미 변형 존재.
- 관련 레슨: `selection-lasso`

**마술봉 (Magic Wand)**
- 쉬운 설명: 비슷한 색 영역을 한 번에 선택해주는 도구예요.
- 정식 정의: 허용치(tolerance) 내 유사 색상 픽셀을 선택.
- 관련 레슨: `selection-magic-wand`

**빠른 선택 (Quick Selection)**
- 쉬운 설명: 브러시로 문지르듯 드래그하면 영역을 알아서 따내는 도구예요.
- 정식 정의: 에지 감지 기반 선택 도구. 마술봉보다 영리함.
- 관련 레슨: `selection-magic-wand`

**페더 (Feather)**
- 쉬운 설명: 선택 영역의 가장자리를 부드럽게 흐리는 효과예요.
- 정식 정의: 선택 경계에 페더 반경만큼 그라디언트 적용.
- 관련 레슨: `selection-modify`

**펜 툴 (Pen Tool)**
- 쉬운 설명: 점과 선으로 정교한 모양을 그리는 도구예요. 마스터하면 포토샵 고수가 됩니다.
- 정식 정의: 베지어 곡선 기반 패스 생성 도구.
- 관련 레슨: `pen-tool-straight`, `pen-tool-curve`, `pen-tool-practice`

**앵커 포인트 (Anchor Point)**
- 쉬운 설명: 펜 툴로 찍은 점이에요. 이 점들이 이어져 선이나 곡선이 됩니다.
- 정식 정의: 베지어 패스의 제어점.
- 관련 레슨: `pen-tool-straight`

**핸들 (Handle)**
- 쉬운 설명: 앵커 포인트에서 뻗어나오는 막대기예요. 곡선의 휨 정도를 조절해요.
- 정식 정의: 베지어 곡선의 방향 제어점.
- 관련 레슨: `pen-tool-curve`

**패스 (Path)**
- 쉬운 설명: 펜 툴로 그린 선이에요. 눈에 보이지 않는 가상의 선으로, 선택 영역이나 모양으로 변환할 수 있어요.
- 정식 정의: 앵커 포인트로 구성된 벡터 경로.
- 관련 레슨: `pen-tool-straight`, `pen-tool-practice`

### 변형과 이동 (tool)

**자유변형 (Free Transform)**
- 쉬운 설명: 이미지의 크기, 회전, 기울기를 자유롭게 바꾸는 기능이에요. Ctrl+T예요.
- 정식 정의: 선택 레이어/영역에 대해 scale, rotate, skew, distort, perspective 변형을 적용.
- 관련 레슨: `free-transform`

**뒤틀기 (Warp)**
- 쉬운 설명: 이미지를 고무찰흙처럼 구부리고 일그러뜨리는 기능이에요.
- 정식 정의: 격자 기반 비선형 변형.
- 관련 레슨: `warp-perspective`

### 색 (color)

**전경색 (Foreground Color)**
- 쉬운 설명: 지금 붓에 묻은 색이에요. 브러시나 페인트통으로 칠할 때 이 색이 나와요.
- 정식 정의: 기본 도구가 사용하는 주 색상. 도구상자 하단 사각형 위쪽.
- 관련 레슨: `foreground-background-color`

**배경색 (Background Color)**
- 쉬운 설명: 예비로 준비된 색이에요. 지우개로 배경 레이어를 지우면 이 색이 나와요.
- 정식 정의: 보조 색상. 일부 도구와 필터가 사용.
- 관련 레슨: `foreground-background-color`

**스포이드 (Eyedropper)**
- 쉬운 설명: 이미지에서 원하는 색을 "뽑아내" 전경색으로 쓰는 도구예요.
- 정식 정의: 캔버스의 픽셀 색상을 샘플링하는 도구.
- 관련 레슨: `foreground-background-color`

**그라디언트 (Gradient)**
- 쉬운 설명: 색이 점점 바뀌는 효과예요. 노을처럼 빨강에서 주황으로 서서히 바뀌는 거죠.
- 정식 정의: 두 개 이상의 색 사이의 점진적 전환.
- 관련 레슨: `gradient-basics`

**색조 (Hue)**
- 쉬운 설명: 색의 종류예요. 빨강, 파랑, 노랑 같은.
- 정식 정의: HSB 색 공간의 H. 색상환에서의 각도(0-360).
- 관련 레슨: `hue-saturation`

**채도 (Saturation)**
- 쉬운 설명: 색이 얼마나 진한지예요. 0이면 흑백, 100이면 제일 쨍한 색.
- 정식 정의: 색의 순도. HSB의 S.
- 관련 레슨: `hue-saturation`

**명도 (Brightness/Lightness)**
- 쉬운 설명: 색이 얼마나 밝은지예요. 0은 검정, 100은 흰색.
- 정식 정의: 색의 밝기. HSB의 B 또는 HSL의 L.
- 관련 레슨: `hue-saturation`, `levels-curves`

### 조정과 효과 (effect)

**조정 레이어 (Adjustment Layer)**
- 쉬운 설명: 사진 위에 씌우는 색안경 같아요. 원본은 안 바뀌고 보기만 바뀌어요.
- 정식 정의: 하위 레이어들에 비파괴적으로 적용되는 색/톤 보정 레이어.
- 관련 레슨: `adjustment-layer-intro`, `levels-curves`

**레벨 (Levels)**
- 쉬운 설명: 그림자, 중간톤, 밝은 부분의 밝기를 조절하는 도구예요. 히스토그램을 보고 조절해요.
- 정식 정의: 이미지의 톤 범위를 재매핑하는 기능.
- 관련 레슨: `levels-curves`

**커브 (Curves)**
- 쉬운 설명: 레벨보다 더 섬세하게 밝기를 조절하는 도구예요. 곡선 그래프로 제어해요.
- 정식 정의: 입력 톤과 출력 톤의 관계를 곡선으로 제어.
- 관련 레슨: `levels-curves`

**히스토그램 (Histogram)**
- 쉬운 설명: 이미지에 밝은 픽셀과 어두운 픽셀이 얼마나 있는지 보여주는 막대 그래프예요.
- 정식 정의: 픽셀 밝기 분포 시각화.
- 관련 레슨: `levels-curves`

**레이어 스타일 (Layer Style)**
- 쉬운 설명: 레이어에 그림자, 외곽선 같은 효과를 한번에 주는 기능이에요.
- 정식 정의: 레이어에 적용되는 효과 모음. 드롭 섀도우, 스트로크, 광택 등.
- 관련 레슨: `layer-style-basics`, `layer-style-shadow-stroke`

**그림자 효과 (Drop Shadow)**
- 쉬운 설명: 레이어 뒤에 그림자를 만드는 효과예요.
- 정식 정의: 레이어 뒤쪽으로 드리워지는 그림자 효과.
- 관련 레슨: `layer-style-shadow-stroke`

**외곽선 (Stroke)**
- 쉬운 설명: 레이어 테두리에 선을 두르는 효과예요.
- 정식 정의: 레이어 경계를 따라 적용되는 선.
- 관련 레슨: `layer-style-shadow-stroke`

### 보정 도구 (tool)

**닷지 (Dodge)**
- 쉬운 설명: 문지르면 그 부분이 밝아지는 도구예요. 사진에서 얼굴 하이라이트 줄 때 써요.
- 정식 정의: 선택 영역을 밝게 만드는 도구. 아날로그 암실의 dodging에서 유래.
- 관련 레슨: `dodge-burn`

**번 (Burn)**
- 쉬운 설명: 문지르면 그 부분이 어두워지는 도구예요. 그림자를 강조할 때 써요.
- 정식 정의: 선택 영역을 어둡게 만드는 도구.
- 관련 레슨: `dodge-burn`

**힐링 브러시 (Healing Brush)**
- 쉬운 설명: 잡티를 주변 피부 느낌에 맞춰 지워주는 똑똑한 도구예요.
- 정식 정의: 샘플 영역의 질감은 유지하면서 색/톤을 주변에 맞추는 복원 도구.
- 관련 레슨: `skin-retouch-basics`

**복제 스탬프 (Clone Stamp)**
- 쉬운 설명: 원하는 부분을 그대로 복사해서 다른 곳에 붙이는 도구예요.
- 정식 정의: 샘플 픽셀을 다른 영역에 복제.
- 관련 레슨: `skin-retouch-basics`

---

### 고급 (advanced)

**스마트 오브젝트 (Smart Object)**
- 쉬운 설명: 원본 그림을 유리 액자에 넣어둔 것 같아요. 언제든 꺼내서 수정할 수 있어요.
- 정식 정의: 원본 데이터를 보존하며 비파괴 편집을 가능하게 하는 레이어 형식.
- 관련 레슨: `smart-object-intro`, `smart-object-filter`

**스마트 필터 (Smart Filter)**
- 쉬운 설명: 스마트 오브젝트에 적용되는 필터예요. 나중에 강도를 바꾸거나 지울 수 있어요.
- 정식 정의: 스마트 오브젝트에 비파괴적으로 적용되는 필터.
- 관련 레슨: `smart-object-filter`

**채널 (Channel)**
- 쉬운 설명: 빨강/초록/파랑 색을 따로 분리한 것이에요. 정밀 합성에 써요.
- 정식 정의: 색상 정보를 분리 저장하는 흑백 이미지. RGB 모드엔 R/G/B 채널.
- 관련 레슨: `channel-mask`

**알파 채널 (Alpha Channel)**
- 쉬운 설명: 투명도 정보만 저장하는 채널이에요.
- 정식 정의: 이미지의 불투명도 정보를 담은 흑백 채널.
- 관련 레슨: `channel-mask`

**액션 (Action)**
- 쉬운 설명: 반복 작업을 녹화해놓고 버튼 하나로 재생하는 기능이에요.
- 정식 정의: 일련의 편집 단계를 기록해 재실행 가능하게 만든 매크로.
- 관련 레슨: `action-basics`, `batch-process`

**배치 처리 (Batch Processing)**
- 쉬운 설명: 여러 파일에 같은 작업을 한 번에 적용하는 기능이에요. 100장 리사이즈 5초에 끝나요.
- 정식 정의: 지정한 액션을 여러 파일에 일괄 적용.
- 관련 레슨: `batch-process`

**목업 (Mockup)**
- 쉬운 설명: 디자인을 실제 제품에 입혀놓은 것처럼 보여주는 이미지예요. 앱 화면을 아이폰 사진 위에 얹는 거죠.
- 정식 정의: 최종 디자인을 실제 매체/제품 컨텍스트에서 시각화한 이미지.
- 관련 레슨: `mockup-basics`, `mockup-create`, `mockup-use`

**카메라 RAW (Camera RAW)**
- 쉬운 설명: 디지털카메라의 원본 파일을 편집하는 포토샵 기능이에요. JPG보다 훨씬 세밀하게 조정할 수 있어요.
- 정식 정의: 가공되지 않은 센서 데이터를 편집하는 Adobe의 RAW 처리 모듈.
- 관련 레슨: `camera-raw-intro`, `camera-raw-color`

**색 프로파일 (Color Profile)**
- 쉬운 설명: 이 이미지는 어떤 색 규칙으로 만들어졌는지 알려주는 꼬리표예요. sRGB, Adobe RGB 같은 것.
- 정식 정의: 기기 간 색 재현 일관성을 위한 ICC 프로파일.
- 관련 레슨: `color-space-print`

**아트보드 (Artboard)**
- 쉬운 설명: 한 문서 안에 여러 개의 캔버스를 만드는 기능이에요. 앱 화면 여러 개를 한 파일에 담을 때 써요.
- 정식 정의: 단일 PSD 내에서 독립적 좌표계를 가진 여러 캔버스 영역.
- 관련 레슨: (Level 3에서 선택적 언급)

**가이드 (Guide)**
- 쉬운 설명: 캔버스에 보이는 도우미 선이에요. 실제 이미지엔 안 찍혀요.
- 정식 정의: 편집 보조용 수평/수직 기준선. 출력물에 포함되지 않음.
- 관련 레슨: (Level 2 포스터 프로젝트에서 언급)

**그리드 (Grid)**
- 쉬운 설명: 캔버스에 바둑판처럼 찍히는 격자선이에요. 정렬 도우미예요.
- 정식 정의: 일정 간격의 기준 격자. 스냅 기능과 함께 사용.
- 관련 레슨: (Level 2 포스터 프로젝트에서 언급)
