## 🎯 오늘의 목표

포토샵에서 가장 자주 쓰는 "화면 움직이기" 기술을 익혀요. 단축키
4개만 손에 붙이면 작업 속도가 눈에 띄게 빨라져요.

## 🍎 일상 비유로 먼저 이해하기

이미지를 보는 건 지도를 보는 것과 같아요. **돋보기**로 한 부분을
크게 들여다보고, **손바닥**으로 지도를 좌우로 밀어가며 보죠.
확대한 후엔 어디로 갈지 안 보이니 이 두 도구를 번갈아 쓰는 게
포토샵의 일상이에요.

단축키 몇 개가 이걸 1초 안에 해결해줘요. 오늘은 손가락에 단축키를
붙이는 레슨이에요.

## 📖 핵심 개념

### 돋보기 도구(Zoom Tool)

- 단축키 **Z**.
- 캔버스를 클릭하면 그 지점을 중심으로 확대돼요.
- **Alt(Mac: Option)** 를 누른 채 클릭하면 축소.

### 손 도구(Hand Tool)

- 단축키 **H**.
- 확대한 상태에서 캔버스를 **드래그**해 화면을 좌우/상하로 이동.

### 스페이스바 + 드래그 (가장 많이 써요)

어떤 도구를 쓰고 있든 **스페이스바를 누른 채 드래그** 하면 잠시
손 도구로 바뀌어요. 손을 뗀 순간 원래 도구로 돌아와요.

<Callout type="tip" title="실전에선 스페이스바가 90%">
돋보기와 손 도구를 툴바에서 직접 고르는 일은 거의 없어요. 단축키
**스페이스바 + 드래그** 로 다 해결돼요.
</Callout>

### 자주 쓰는 확대/축소 단축키

| 동작 | Windows | Mac |
|---|---|---|
| 확대 | Ctrl + `+` | Cmd + `+` |
| 축소 | Ctrl + `-` | Cmd + `-` |
| 화면 맞춤 | Ctrl + 0 | Cmd + 0 |
| 100% 실제 크기 | Ctrl + 1 | Cmd + 1 |
| 확대(마우스 휠) | Alt + 스크롤 | Option + 스크롤 |

<Callout type="warning" title="100%로 확인하는 습관">
SNS나 웹용 이미지는 100% 확대율로 봐야 실제 보이는 크기예요.
작업 중엔 크게 보다가 저장 전에 **Ctrl+1** 로 확인해요.
</Callout>

### 확대율 표시

포토샵 창의 **왼쪽 아래 상태 표시줄** 또는 **제목 탭**에 지금
확대율(%)이 나와요. 숫자를 직접 입력해 정확한 확대율로 갈 수도
있어요.

## 😵 이게 안 돼요 (자주 묻는 문제)

- **Q**: Alt + 휠이 안 먹어요.
  **A**: **편집 > 환경설정 > 도구** 에서 **"스크롤 휠로 확대"**
  옵션을 켜면 Alt 없이도 휠로 확대할 수 있어요.

- **Q**: 확대가 너무 빠르게 또는 느리게 돼요.
  **A**: 돋보기 도구 선택 시 옵션바의 **"움직이는 확대/축소
  (Scrubby Zoom)"** 를 켜면 드래그로 부드럽게 조절돼요.

- **Q**: 화면이 흐릿하게 보여요.
  **A**: 극단적 확대에서는 픽셀이 보이는 게 정상이에요. **Ctrl+1**
  로 100%로 돌아가면 원래 화질로 보여요.
