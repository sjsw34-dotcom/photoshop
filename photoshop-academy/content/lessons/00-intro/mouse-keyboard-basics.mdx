## 🎯 오늘의 목표

포토샵에서 손가락이 가장 자주 하는 일을 익힐 거예요. 클릭 세 종류와
수정 키(Modifier Key) 세 개만 알면 거의 모든 작업의 시작이 편해져요.

## 🍎 일상 비유로 먼저 이해하기

**Shift 키**는 "똑바로 해줘"라고 말해주는 키예요. 원을 그릴 때
Shift를 누르면 정원이 되고, 선을 그을 때 누르면 수평선이나
수직선만 나와요.

**Alt 키(Mac은 Option)** 는 "하나 더 복사해줘"라고 말해주는 키예요.
레이어를 Alt 누른 채 드래그하면 원본은 그대로 두고 복사본이
생겨요.

**Ctrl 키(Mac은 Cmd)** 는 "명령을 실행해줘"라고 말하는 키예요.
거의 모든 단축키의 출발점이에요.

## 📖 핵심 개념

### 마우스 동작 세 가지

- **클릭(Click)**: 한 번 누르기. 도구 선택, 버튼 누르기에 써요.
- **더블클릭(Double Click)**: 빠르게 두 번. 레이어 이름 바꾸기,
  폴더 열기 같은 "더 깊게 들어가기"에 써요.
- **드래그(Drag)**: 누른 채 끌기. 이동, 선택 영역 만들기,
  크기 바꾸기에 써요.

### 수정 키 세 가지

수정 키는 다른 동작 위에 "같이 눌러서" 효과를 바꾸는 키예요.

| 키 | 역할 | 예시 |
|---|---|---|
| Shift | 비율 유지, 직선, 중복 선택 | Shift+드래그로 정원 그리기 |
| Alt / Option | 복사, 임시 반전 | Alt+드래그로 레이어 복사 |
| Ctrl / Cmd | 명령 실행 | Ctrl+S 저장 |

<Callout type="tip" title="두 개 같이 눌러도 돼요">
Ctrl+Shift+S(다른 이름으로 저장)처럼 수정 키 여러 개를 함께 쓰는
경우가 많아요. 수정 키끼리는 서로 싸우지 않아요.
</Callout>

### 우클릭 메뉴

우클릭(Mac은 Control+클릭 또는 트랙패드 두 손가락 탭)은 **"지금 이
위치에서 할 수 있는 일"** 을 보여주는 작은 메뉴예요. 실수로 눌러도
그냥 ESC로 닫으면 돼요.

### 스페이스바 + 드래그

화면에 이미지가 너무 커서 한 번에 안 보일 때 쓰는 마법이에요.
**스페이스바를 누른 채** 마우스를 드래그하면 화면이 따라 움직여요.

## 😵 이게 안 돼요 (자주 묻는 문제)

- **Q**: Alt를 누르고 드래그했는데 복사가 안 돼요.
  **A**: 이동 도구(V) 상태에서 해봐요. 다른 도구 상태에서는 Alt가
  다른 역할(스포이드 전환 등)을 하기도 해요.

- **Q**: Mac인데 Alt 키가 어디 있어요?
  **A**: Mac의 **Option** 키가 Alt와 같아요. Cmd 옆에 있어요.

- **Q**: 우클릭이 안 돼요(노트북 트랙패드).
  **A**: 트랙패드 **두 손가락 탭** 으로 우클릭이에요. 설정에서
  꺼져 있을 수 있으니 시스템 설정도 확인해요.

- **Q**: 스페이스바를 눌러도 손 모양 커서가 안 떠요.
  **A**: 문자 입력 도구(T)를 쓰고 있을 땐 스페이스가 띄어쓰기가
  돼요. 다른 도구로 전환한 뒤 다시 해봐요.
